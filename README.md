# 2026CsieCamp-FakeWeb
This is for educational purposes only for NTNU CSIE Camp. (僅供師大資工營教學使用)

## Assets Structure (資源結構)

`資訊工程學系_files/` 已整理為下列結構：

- `資訊工程學系_files/樣式/`：所有 `.css` 檔案
- `資訊工程學系_files/腳本/`：所有 `.js` 與 `js` 腳本檔案
- `資訊工程學系_files/圖片/`：頁面使用的圖片檔（`.png/.jpg/.bmp/...`）

`index.html` 內的資源路徑已同步更新為新結構。

## Maintenance Note (維護提醒)

- 新增或移動資源檔案後，請同步更新 `index.html` 對應路徑。
- 若要再進一步整理（例如依套件分群），建議先做整站路徑檢查，避免壞連結。
